List of the full stack web development tech stack

1.HTML

Primary Use Cases Key Features/ Benefits
Building the structure of the webpages
  • Simple and easy to learn
  • Compatible with all web browsers
  • Allow for semantic markup

2.CSS

Primary Use Cases Key Features/ Benefits
Selectively style HTML elements.
  • Specifies how HTML elements should be displayed on screens.
  • Includes styling rules which are interpreted by the client browser and applied to various elements..
  • Can control the color text, the style of fonts, spacing between elements, how columns are sized, variation in display for different devices and screen size as well as a variety of other effects.

3.JS

Primary Use Cases Key Features/ Benefits
Makes web pages more interactive and dynamic
  • Platform independence
  • Dynamic typing
  • Form checks and calculations
